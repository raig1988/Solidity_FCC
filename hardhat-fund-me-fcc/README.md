<!-- USAGE EXAMPLES -->
## Usage

In this project, we can test deployment to a local network or local host and also test deployment into Ethereum's goerli testnet.
Testing has been set by creating unit tests for local testing and also staging test for deployed contracts.
Scripts to allow funding and withdraw of the funds have been written under the ./scripts folder.
In hardhat.config.js, gas reporter is set so we are able to optimize gas consumption inside the contract.
